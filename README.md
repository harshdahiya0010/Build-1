# 🤖 Haro AI

An intelligent AI chatbot built using **Python**, **LangChain**, **OpenAI GPT**, and **Streamlit**.

Haro AI is a conversational assistant capable of maintaining chat history and generating intelligent responses through Large Language Models (LLMs).

---

## 🚀 Features

- 💬 Interactive AI conversations
- 🧠 Chat history memory
- ⚡ Powered by OpenAI GPT
- 🔗 Built with LangChain
- 🌐 Streamlit web interface
- 🔒 Secure API key using .env
- 📦 Simple and lightweight project

---

## 🛠 Tech Stack

- Python
- Streamlit
- LangChain
- OpenAI
- python-dotenv

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Haro-AI.git
```

Move inside the project

```bash
cd Haro-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a .env file

```
OPENAI_API_KEY=your_api_key
```

Run the application

```bash
streamlit run app.py
```
